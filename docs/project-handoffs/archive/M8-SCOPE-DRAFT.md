## 1. 立项依据（差距盘点结论）

对照 Raft PRD 逐条盘点（2026-07-14）：

- **已有且更强**：held-draft 可见+放行（对方黑洞）、任务状态机角色权限+T7 硬关口（对方无权限靠自觉）、画布 DAG 编排（对方无画布）、技能白名单（对方全量继承）、worktree/Diff/预览/部署/成本账（对方无代码工作面）。
- **真差距**：多用户登录邀请（→M9+）、建 Agent 通用 UI 入口、onboarding 体验、runtime 广度（2 vs 9，押后）、移动端/生态（押后）。
- **自家账**：真机实测暴露的画布/前端缺陷 + CR-M8 同族残留。

结论：M8 不抄 Raft 功能清单，只取「单机产品化外壳」两三件；主力投给自家差异化面的缺陷修复与体验打磨。
