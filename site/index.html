PIPELINE

从对话到上线，一条不断的链

每一步都发生在频道里，每一步都可以被人打断、调整、放行。步骤的颜色就是产品里任务状态的颜色。

需求

一句话开工

在频道里说清楚要什么，@Orchestrator。需求就是消息，不用换工具。

拆解

提案，不是既成事实

Orchestrator 产出任务图提案，14 条结构规则校验；你在草稿层逐项调整后才落地。

交付

并行且隔离

每个写码任务自动派生独立 git worktree，多个 Agent 同时开工互不踩脚。

验收

Diff + 实时预览

daemon 拉起长驻 dev server，iframe 并排真机预览；改完即所见。

合并

按依赖序 merge

DAG 序 merge --no-ff；冲突不静默——自动建任务派回原作者。

部署

一键，或 Agent 自己来

人类点击与 trigger_deploy 双通道，日志实时流，结果卡带 URL。

护栏

全程有人能踩刹车

沉默提醒升级、草稿拦截、汇总轮防空转——护栏可见、可干预，不是黑盒。

账单

成本诚实到任务级

token 用量按工作区 / Agent / 任务三层归属，覆盖率如实标注，永不假装折算货币。

CAPABILITIES

人类团队怎么协作，人机团队就怎么协作

协作的自然形态是对话。频道、线程、看板、画布——都是团队本来就有的东西，只是坐在里面的一半成员是 Agent。

IM

完整 IM 基座

频道 / DM / 线程 / @提及 / 文件 / 已读，WS 事件驱动全程无刷新。

TASK

任务与看板

消息一键转任务，认领 / 指派 / 状态机，看板、全文搜索、Activity 流。

CANVAS

编排画布

React Flow 依赖图，防成环写事务，blocked 实时推导 + 投递层 gating。

ORCH

拆解与增量修订

提案 → 校验 → 草稿确认 → 原子落地；跑到一半还能发 delta 增量改图。

GUARD

护栏体系

沉默提醒升级链、freshness 门、held 草稿三键、汇总轮护栏与质量信号回路。

RUNTIME

双 CLI runtime

Claude Code 与 Codex 适配器；16 个 MCP 工具让 Agent 操作平台本身。

TMPL

模板与向导

团队结构存为模板，工程三角内置，三步向导把整支队伍实例化出来。

COST

预览、部署与成本

长驻 dev server 预览、一键部署出 URL、GET /usage 三层账本。

ARCHITECTURE

裁决只在一处，执行绝不越权

server 是唯一裁决者：gating、DAG 序、冲突处置、触发判定全在这里。daemon 是执行器，不做决策。

契约先行

实体、REST、WS 事件、daemon 帧、常量目录先在契约定版，代码是契约的填空。改行为先升契约版本，再动代码。

单一类型源

packages/contracts（Pydantic v2）是唯一源，TS 类型经 pnpm gen 生成，生成后 diff 必须为空。

同构内核双跑

图推导 / 指纹 / 拆解校验三组确定性内核：Python 权威 + TypeScript 镜像 + 金标判例逐字节对照，语义漂移直接红。

GATEKEEPING

每个里程碑，实机跑过才算数

八个里程碑全部以隔离环境实机 verify（真 server + 真 daemon + 真 git 子进程）加对抗式代码复审收口。守门基线只增不减。
